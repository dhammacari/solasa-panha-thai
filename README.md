# solasa-panha-thai
# โสฬสปัญหา (ฉบับภาษาไทยเข้าใจง่าย)

โปรเจกต์รวบรวมคำถาม-คำตอบทั้ง 16 ข้อของผู้แสวงบุญในสมัยพุทธกาล โดยเน้นการใช้ภาษาที่เรียบง่าย สบายตา เพื่อให้คนรุ่นใหม่และผู้เริ่มต้นเข้าถึงหัวใจของธรรมะได้ทันที

### จุดเด่นของโปรเจกต์
- **Minimalist Design:** สะอาด เรียบง่าย ไม่ซับซ้อน
- **Dark Mode:** ถนอมสายตาด้วยโทนสีมืดและตัวอักษรสีทอง
- **Easy Language:** แปลจากบาลีเป็นภาษาไทยที่ใช้ในชีวิตประจำวัน
